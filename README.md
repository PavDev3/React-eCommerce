## 📝 Project Summary

- [**src**](src): Contains the main source code of the JavaScript project.
- [**src/components**](src/components): Houses reusable UI components used throughout the application.
- [**src/context**](src/context): Provides a centralized location for managing global state using React Context API.
- [**src/hooks**](src/hooks): Holds custom hooks that encapsulate reusable logic for functional components.
- [**src/reducers**](src/reducers): Contains Redux reducers for managing application state.
- [**src/assets**](src/assets): Stores static assets such as images, fonts, and stylesheets.
- [**src/mocks**](src/mocks): Contains mock data used for testing and development purposes.
- [**public**](public): Contains the static files that will be served by the web server.


## 💻 Stack

- [react](https://reactjs.org/): A JavaScript library for building user interfaces.
- [react-dom](https://reactjs.org/docs/react-dom.html): Provides DOM-specific methods that can be used at the top level of your app.
- [vite](https://vitejs.dev/): A fast build tool that significantly improves the frontend development experience.
- [eslint](https://eslint.org/): A pluggable and configurable linter tool for identifying and reporting on patterns in JavaScript.
- [eslint-plugin-react](https://www.npmjs.com/package/eslint-plugin-react): React specific linting rules for ESLint.
- [eslint-plugin-react-hooks](https://www.npmjs.com/package/eslint-plugin-react-hooks): Lint rules for React hooks.
- [vitejs/plugin-react-swc](https://github.com/vitejs/vite/tree/main/packages/plugin-react-refresh): A plugin for Vite that uses SWC to transform React code.
- [types/react](https://www.npmjs.com/package/@types/react): TypeScript type definitions for React.

These eight dependencies cover crucial aspects like React for building user interfaces, DOM manipulation, fast development experience, linting rules, and TypeScript type definitions.

## 🚀 Run Locally
1.Clone the React-eCommerce repository:
```sh
git clone https://github.com/PavDev3/React-eCommerce
```
2.Install the dependencies with one of the package managers listed below:
```bash
pnpm install
bun install
npm install
yarn install
```
3.Start the development mode:
```bash
pnpm dev
bun dev
npm run dev
yarn dev
```
